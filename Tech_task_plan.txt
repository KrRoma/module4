Например:
---------

Данный проект называется Посоветуй_Игру. Идея проекта заключается в реализации программы для поиска названий игр в зависимости от жанра. Проект написан для упрощения пользователю поиска игры выбранного жанра.

Для получения названия игры по названию жанра пользователь должен выполнить следующие шаги:
	- запустить программу
	- нажать кнопку с интересующим его жанром
	- получить название игры
Для закрытия программы пользователь должен выполнить следующие шаги:
	- нажать кнопочку крестик
	
Для разработки использовался язык программирования Python версии 3.8, а также библиотеки:
	- BeautifulSoup, предназначеенная для анализа документов HTML и XML
	- random, предоставляющая функции для генерации случайных чисел или выбора случайного элемента последовательности
В качестве среды разработки использовался PyCharm

Код сожержит основную часть и функции quest_click и arcada_click. 
Основная часть кода создаёт окно приложение и добавляет на него кнопки quest и arcada. При нажатии на кнопку quest вызывается функия quest_click, а при нажатии на кнопку arcada вызывается функия arcada_click.
Функция quest_click делает запрос к сайту "..." и получает информацию по классу "...". Далее из полученной информации функция получает game_names и выбирает одно название с помощью функции choice библиотеки random. Далее функция выводит полученное название игры пользователю в виджет Label.
Функция arcada_click работает аналогично функции quest_click, но делает запрос к сайту "....".
